Projet de recommandation musicale

Système de recommandation basé sur la data et l’IA – 3ème année

Objectif du projet

Ce projet vise à concevoir un système complet de recommandation musicale basé sur l’analyse de données utilisateurs. Il combine data engineering, machine learning et développement logiciel.

Étapes du projet

  • Collecte et analyse de données utilisateurs
  • Conception et création de la base de données
  • Développement de plusieurs algorithmes de recommandation
  • Réalisation d’une API pour exposer les résultats

Technologies utilisées

  • Python (Pandas, NumPy, Scikit-learn)
  • SQL
  • API REST
  • Modèles de recommandation (collaboratif, contenu, hybridation)
